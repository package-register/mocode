Use this scratchpad tool to reason through complex problems before taking action.

Unlike other tools, think does not retrieve new information or change any state.
It echoes the thought back so it remains visible in the conversation context.
Use it when you need to:

- Check which rules or policies apply to the current task
- Verify you have all required information before acting
- Validate a plan step-by-step before executing it
- Break down a complex, multi-step problem into smaller parts
- Avoid mistakes before using destructive or irreversible tools

When in doubt, think first.
